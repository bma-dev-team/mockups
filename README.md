# BMA Dev Team Marketplace Mockups

Public hosting for clickable UI mockups attached to BMA Dev Team proposals on the [BuildMyApp](https://www.appmarketplace.dev) marketplace.

Each subdirectory is the slug of a single proposal; `index.html` is the mockup entrypoint. Hosted via GitHub Pages at `https://ox805.github.io/bma-dev-team-mockups/<slug>/`.

The source mockups live alongside their proposals in the [build_my_app](https://github.com/Ox805/build_my_app) repo at `bma-dev-team/proposals/mockups/<slug>/`. The `scripts/publish_mockup.sh <slug>` script copies them here and pushes.
