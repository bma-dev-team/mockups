# BMA Dev Team Marketplace Mockups

Public hosting for clickable UI mockups attached to BMA Dev Team proposals on the [BuildMyApp](https://www.appmarketplace.dev) marketplace.

Each subdirectory is the slug of a single proposal; `index.html` is the mockup entrypoint. Hosted via GitHub Pages at `https://bma-dev-team.github.io/mockups/<slug>/`.

The source mockups live alongside their proposals in BMA's main repo at `bma-dev-team/proposals/mockups/<slug>/`. The `scripts/publish_mockup.sh <slug>` script copies them here and pushes.
